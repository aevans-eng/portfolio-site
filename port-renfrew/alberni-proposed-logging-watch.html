This is forward-looking: what is coming, not past clearcuts. Overlap and proximity are reasons to look and to comment, not proof of harm. Each block's comment period is set by its Forest Operations Map project; once that window closes you can still file a records request. The map version is at the interactive map (turn on the red Proposed logging layer).
